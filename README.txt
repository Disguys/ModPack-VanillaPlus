⚡ ENG:
How do I start the server on your PC?:
1) Open start.bat file, after which edit parameter -Xms2G -Xmx2G to your available RAM
2) Run file start.bat (If this comes up with an error it means you need Java 16)
3) Accept EULA agreement (eula.txt)
4) Run file start.bat again
5) Profit!

⚡ RU:
Как запустить сервер на вашем ПК?:
1) Откройте файл start.bat, после чего отредактируйте параметр -Xms2G -Xmx2G на ваше доступное количество ОЗУ.
2) Запустите start.bat (если это приведет к ошибке, это означает, что вам нужна Java 16)
3) Примите лицензионное соглашение (eula.txt)
4) Запустите файл start.bat еще раз
5) Профит!
